## Persistencia Básica – Core Data (iOS)

# Proyecto correspondiente al ejercicio Persistencia Básica de la asignatura Persistencia de Datos (Máster UA).

# Descripción

Aplicación iOS desarrollada en Swift que implementa persistencia local utilizando Core Data.

# El proyecto permite:

- Crear y almacenar entidades Nota

- Guardar datos en SQLite mediante Core Data

- Recuperar registros almacenados

- Mostrar resultados en interfaz gráfica

# Tecnologías

- Swift

- UIKit

- Core Data

- Xcode

# Estructura principal

- AppDelegate.swift → Configuración del stack de Core Data

- ViewController.swift → Lógica de interfaz y operaciones CRUD

- .xcdatamodeld → Modelo de datos (Nota)

# Objetivo académico

Comprender el funcionamiento básico de:

- NSManagedObject

- NSManagedObjectContext

NSFetchRequest

Operaciones save, fetch y delete
